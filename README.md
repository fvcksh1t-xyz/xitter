## Installation

### 1. Clone the repository
```bash
git clone https://github.com/fvcksh1t-xyz/xitter
cd xitter
```

### 2. Run the installer

#### Linux
```bash
chmod +x install.sh
./install.sh
```
Then search for "xitter" in your app menu or type `xitter` in terminal.

#### Windows
Double-click `install.bat`
Then search for "xitter" in Start Menu.

#### macOS
```bash
chmod +x install_mac.sh
./install_mac.sh
```
Restart your terminal, then type `xitter`.

## Uninstall

### Linux
```bash
rm -rf ~/.local/share/xitter
rm ~/.local/share/applications/xitter.desktop
rm ~/.local/bin/xitter
```

### Windows
Delete folder: `%APPDATA%\xitter`

### macOS
```bash
rm -rf ~/Library/Application\ Support/xitter
rm ~/.local/bin/xitter
```
